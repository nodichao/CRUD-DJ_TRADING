Site de vente en Ligne

Le but de ce projet est de gérer les fonctionnalites de gestion de stock pour un site de vente en ligne.
Ainsi, l'utilisateur devra se connecter via un login et un mot de passe, il pourra ajouter, supprimer, modifier et lire les informations sur les produits.
Le langage de base de ce projet est PHP, il est accompagné de HTML, CSS, Bootstrap, SQL et le SGBD utilise est MySQL.
